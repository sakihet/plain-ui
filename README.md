# * UI

## Slogan

Plain old CSS is enough.

## Files and their roles

- reset.css
  - Reduce the complexity
- base.css
  - Add basic things
- token.css
  - Manage design tokens by [CSS custom properties](https://developer.mozilla.org/en-US/docs/Web/CSS/--*)
- utility.css
  - Utility-First CSS
- layout.css
  - Create layouts
- pattern.css
  - Create common UI
